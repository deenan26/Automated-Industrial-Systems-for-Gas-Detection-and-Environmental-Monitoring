# Automated Industrial Systems for Gas Detection and Environmental Monitoring

## 📌 Project Overview
This project presents an automated industrial safety system designed to detect hazardous gas leaks and monitor environmental conditions using an autonomous line-following robot. The system uses an 8051 microcontroller to integrate gas sensors, infrared sensors, and ultrasonic sensors for navigation and monitoring in industrial environments.

The robot continuously scans for dangerous gas levels and environmental changes while following predefined paths. When hazardous gas is detected, the system triggers an alert and sends notifications through a Bluetooth module for remote monitoring.

This solution helps reduce human exposure to hazardous environments and improves safety and efficiency in industrial monitoring applications.

## 🔧 Working Principle
The system is built around the **AT89S52 8051 microcontroller**, which processes data from multiple sensors. IR sensors guide the robot along predefined paths, while ultrasonic sensors detect obstacles to ensure safe navigation.

The MQ gas sensor continuously monitors the surrounding environment for hazardous gases. When gas levels exceed a safe threshold, the system activates a buzzer alert and sends a notification via the Bluetooth module to a connected device. The L298N motor driver controls the robot's DC motors for movement.
